# AI-Based Resume Screening & Ranking System

## Overview
This project is an AI-powered system that automates resume screening and candidate ranking for recruitment processes.

## Problem Statement
Manual resume screening is time-consuming and inefficient for HR teams.

## Proposed Solution
The system uses Artificial Intelligence and Natural Language Processing (NLP) to:
- Extract skills from resumes
- Compare resumes with job descriptions
- Rank candidates automatically
- Generate a shortlist

## Tech Stack
- Python
- NLP
- Machine Learning
- Flask
- MySQL

## Future Scope
- Automated email notifications
- Integration with job portals
- Bias detection in hiring process
